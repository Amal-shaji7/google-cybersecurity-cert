# Data Handling Practices — Least Privilege Analysis

## Overview

This activity applied the principle of least privilege
to a real data leak incident at an educational
technology company. An employee accidentally shared
a link to confidential internal documents with an
external business partner during a sales call and these
documents were then posted publicly on social media.

The analysis identifies the control failures that
led to the incident, reviews NIST SP 800-53 AC-6,
recommends specific control enhancements, and
justifies how those improvements would reduce the
likelihood of recurrence.
