## Key Concepts Covered In This Course

- NIST Cybersecurity Framework (CSF) — five core functions: Identify, Protect, Detect, Respond, Recover
- Security frameworks and controls
- OWASP security principles
- Security audits — purpose, scope, and methodology
- Basic security tools — SIEM, playbooks, packet sniffers
- Risk identification, assessment, and management
- Compliance frameworks — PCI DSS, GDPR, SOC Type 1 and 2
