## Portfolio Project

### IP Allow List Update Algorithm
Developed a Python algorithm to automate the
management of an IP address allow list at a
healthcare organisation. The algorithm reads
the current allow list from a file, cross-
references it against a remove list of
unauthorised IP addresses, removes any matches,
and writes the updated list back to the file.
Demonstrates file handling, string and list
manipulation, iterative logic, and conditional
evaluation — all applied to a real access
control security task.
