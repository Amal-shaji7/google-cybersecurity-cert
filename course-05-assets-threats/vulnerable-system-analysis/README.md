# Vulnerable System Analysis
## E-Commerce Database Server Vulnerability Assessment

## Overview

A publicly accessible database server at an e-commerce
company had been open to the internet since the
company's launch three years prior. This portfolio
activity involved conducting a full vulnerability
assessment of the server, identifying threat sources
and events, scoring risks by likelihood and severity,
and proposing a structured remediation strategy to
communicate the risks to business decision makers.

The assessment follows **NIST SP 800-30 Rev. 1**
guidance for risk analysis of information systems.
