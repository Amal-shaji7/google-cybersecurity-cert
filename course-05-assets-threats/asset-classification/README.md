# Home Network Asset Classification

## Overview

Asset inventory exercise identifying devices connected 
to a small home-based business network. Each asset was 
documented by network access type, owner, location, 
and assigned a sensitivity classification — Restricted, 
Confidential, Internal-only, or Public — based on the 
data it handles and the access it provides.

This activity applies the asset management principles 
of the NIST CSF Identify function — establishing 
visibility across connected devices before appropriate 
protections can be applied.

| Classification | Access Level |

| Internal-only | Users on-premises |
