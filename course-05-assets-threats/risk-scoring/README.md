# Risk Scoring — Likelihood and Severity Assessment

## Overview

Risk assessment is a core function of any security 
programme. Before resources can be allocated to 
protective controls, security teams must evaluate 
which risks are most likely to occur and which would 
cause the most damage if they did.

This activity involved evaluating five identified 
risks to a commercial bank's funds using a structured 
risk register. Each risk was scored by likelihood 
of occurrence and severity of impact, then combined 
using a risk priority formula to produce an overall 
score that guides remediation priority.

## Risk Priority Formula
```
Likelihood × Severity = Priority Score
```

| Score | Likelihood | Severity |
|-------|-----------|---------|
| 1 | Rare | Low impact |
| 2 | Likely | Moderate impact |
| 3 | Certain | Catastrophic impact |
