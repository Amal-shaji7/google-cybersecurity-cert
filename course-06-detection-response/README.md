# Course 6 — Sound the Alarm: Detection and Response

## Overview

This folder contains the portfolio deliverable
completed during **Course 6: Sound the Alarm:
Detection and Response** of the Google
Cybersecurity Professional Certificate.

This course covered the full incident response
lifecycle from detection and analysis through
containment, eradication, and recovery. Practical
activities involved using industry-standard tools
including Wireshark, tcpdump, Suricata, Splunk,
and Chronicle to detect, investigate, and respond
to security incidents.

The primary deliverable is an **Incident Handler's
Journal** — a structured record of security
incidents investigated throughout the course,
documenting the 5 W's, tools used, and analytical
findings for each entry.

## Key Concepts Covered

### Incident Response Lifecycle
- NIST Incident Response Lifecycle — Preparation,
Detection and Analysis, Containment Eradication
and Recovery, Post-Incident Activity
- Incident documentation and the 5 W's framework
- Escalation procedures and ticket management

### Detection and Analysis
- Network traffic analysis using Wireshark
and tcpdump
- Packet capture and inspection
- File hash analysis and threat intelligence
lookup using VirusTotal
- IDS/IPS alerts — Suricata signatures and
severity levels
- SIEM log querying and correlation — Splunk
and Chronicle

### Threat Intelligence
- Malware identification through SHA-256
file hash analysis
- Threat actor attribution
- Indicators of Compromise (IOCs)
- Command and Control (C2) communication
identification

### Incident Types Investigated
- Ransomware attack via phishing email
- Malicious file hash — Flagpro malware linked
to BlackTech threat actor
- Phishing email delivering malware attachment
- Suspicious network activity detected through
SIEM and IDS tools
